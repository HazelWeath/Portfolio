Infrastructure Réseau PME

SAE 2.01 • Conception, Déploiement & Sécurisation

Le Besoin (Situation)

Une PME souhaitait refondre son infrastructure réseau pour intégrer une zone publique (DMZ) et sécuriser ses données internes, tout en assurant une connectivité IPv4/IPv6 via un FAI simulé.

Réalisations Techniques (Actions)

  • Segmentation (VLANs) : Isolation des flux (Admin, User, VoIP, Serveurs) pour limiter les domaines de diffusion.
  • Accès aux équipements : Mise en place de SSH sur les switchs et routers afin de pouvoir tout gérer depuis le pc Admin.
  • Sécurité (ACLs) : Filtrage strict des flux inter-VLANs et protection de la DMZ (Stateful Inspection simulée).
  • Services Réseaux : Déploiement de DHCP Relay, DNS Split-Horizon et NAT/PAT.

Compétences du Référentiel

Administrer (Niv. 2) Sécuriser (Niv. 2)

Projet : NaïmCipher

Exploration pratique de la cryptographie symétrique

Fonctionnement Technique

L'algorithme repose sur trois couches de traitement pour garantir la confidentialité et la réversibilité :

  • Cœur XOR (Exclusive OR) : Comparaison binaire bit à bit. C'est l'opération magique qui rend le chiffrement réversible (Message ^ Clé = Chiffré et inversement).
  • Clé Tournante (Rolling Key) : Utilisation du Modulo pour répéter la clé si le message est plus long qu'elle.
  • Encodage Base64 : Conversion du résultat binaire en caractères imprimables pour le web.

Full-Stack & Compatibilité

Le projet démontre une interopérabilité totale :

  • Backend : Script Python (3.8+).
  • Frontend : Interface web en Vanilla JS (Zéro dépendance) hébergée sur Netlify.
